# ShopSphere Enterprise Analytics

## Project Overview

This project demonstrates an end-to-end Business Intelligence solution for a fictional e-commerce company (ShopSphere) using the Olist public dataset.

The project follows the complete analytics lifecycle, including:

- Business Requirement Gathering
- Data Modeling
- Excel Data Cleaning
- PostgreSQL Database
- SQL Business Analysis
- Exploratory Data Analysis
- KPI Design
- Power BI Dashboard
- Business Recommendations

Tools Used

- Excel
- PostgreSQL
- Power BI
- GitHub
- Draw.io
- Jira (Simulation)
- Miro (Simulation)
