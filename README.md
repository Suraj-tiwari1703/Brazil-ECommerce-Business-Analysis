# Brazil E-Commerce Business Analysis

## Project Overview

This project presents an end-to-end Business Intelligence and Business Analysis solution using the Brazilian E-Commerce (Olist) public dataset. It demonstrates the complete analytics lifecycle, from understanding business requirements to delivering actionable insights through interactive Power BI dashboards.

The project covers the following phases:

* Business Requirement Gathering
* Project Planning & Documentation
* Data Understanding
* Data Cleaning & Preparation
* Data Modeling (ER Diagram)
* PostgreSQL Database Design
* SQL Data Analysis & Exploratory Data Analysis (EDA)
* KPI Identification & Business Metrics
* Interactive Power BI Dashboard Development
* Business Insights & Recommendations
